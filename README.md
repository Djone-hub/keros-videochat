# Keros VideoChat

Простой видеочат со скриншарингом для kerosmods.ru

## Функции
- 📹 Видео-звонки (WebRTC P2P)
- 💬 Текстовый чат
- 🖥️ Демонстрация экрана
- 🔗 Комнаты по ссылке
- 👥 Несколько участников

## Установка

```bash
cd videochat
npm install
```

## Запуск

```bash
npm start
```

Сервер запустится на порту 3000 (или PORT из env)

## Деплой на kerosmods.ru

### Вариант 1: Поддомен (chat.kerosmods.ru)
1. Загрузите папку `videochat` на хостинг
2. Установите Node.js зависимости: `npm install`
3. Запустите: `npm start` (или используйте pm2)
4. Настройте Nginx reverse proxy на порт 3000
5. Укажите DNS запись на сервер

### Вариант 2: Подкаталог (kerosmods.ru/chat)
1. Соберите статичную версию (если нужно)
2. Используйте отдельный сервер для WebSocket

### Вариант 3: Отдельный сервер
1. Разверните на VPS (DigitalOcean, Vultr, etc.)
2. Установите Node.js 16+
3. Клонируйте проект
4. Установите PM2: `npm install -g pm2`
5. Запустите: `pm2 start server.js --name keros-videochat`

## Использование

1. Откройте ссылку на чат
2. Введите имя и ID комнаты (или оставьте пустым для создания)
3. Разрешите доступ к камере/микрофону
4. Поделитесь ссылкой для приглашения

## Ссылки
- STUN серверы Google (бесплатно)
- Для production рекомендуется TURN сервер (coturn)

## Требования
- Браузер с поддержкой WebRTC (Chrome, Firefox, Edge, Safari)
- HTTPS для скриншаринга (обязательно в продакшене)
